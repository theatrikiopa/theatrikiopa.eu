# 🎭 theatrikiopa.eu
Η Θεατρική Ομάδα του Οικονομικού Πανεπιστημίου Αθηνών ιδρύθηκε τον Δεκέμβριο του 1979 και πρώτη παράσταση αποτέλεσε η σατιρική κωμωδία “Οι Γραμματιζούμενοι” του Βασίλη Ρώτα που γράφτηκε το 1943. Εντάσσεται από κοινού με άλλες πολιτιστικές ομάδες (χορού, μουσικής κ.ά.) στη Φοιτητική Λέσχη του Πανεπιστημίου απ’ όπου και χρηματοδοτείται.
